## Installation

Install my-project 

```bash
  apt-get install git -y
  git clone https://github.com/GTSivam/Web_adb.git
  cd Web_adb
  chmod 777 install
  chmod 777 run
  sudo ./install
```
## Deployment

To deploy this project run

```bash
  ./run
```
